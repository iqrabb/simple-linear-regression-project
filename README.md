# Simple Linear Regression Project

## Overview

This project implements a Simple Linear Regression model to predict student exam scores based on study hours.

## Dataset

The dataset contains two variables:

* Study Hours (Independent Variable)
* Exam Score (Dependent Variable)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Google Colab

## Project Workflow

1. Import libraries
2. Load dataset
3. Visualize data using scatter plot
4. Split data into training and testing sets
5. Train the Simple Linear Regression model
6. Make predictions
7. Evaluate model performance

## Model Performance

* R² Score: 97.71%

## Regression Equation

Exam Score = 7.1567 × Study Hours + 26.3438

## Example Prediction

If a student studies for 5 hours:

Predicted Exam Score = 62.13
